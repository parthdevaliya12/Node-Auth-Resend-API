# Node Auth Resend API

A secure Node.js Authentication API built with Express.js that provides user registration, login, and email verification using Resend.

## Features

* User Registration
* User Login
* Email Verification
* JWT Authentication
* Password Hashing with bcrypt
* Resend Email Integration
* RESTful API Structure
* Environment Variable Configuration

## Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (JSON Web Token)
* bcryptjs
* Resend API
* dotenv
